# Start Gulp

instalation
-----------
```
$ npm install -save-dev gulp

$ npm install

$ gulp
```


first instalation
-----------
```
$ npm install -g gulp

$ npm install jade --global

$ npm install -save-dev gulp

$ npm install

$ gulp
```
